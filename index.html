Udacity Portfolio

Welcome to my portfolio site. Here you'll find descriptions about the projects I completed as part of my Udacity Full Stack Web Developer Nanodegree program as well as links to their associated GitHub repositories.

This site is a fork of my second project, the responsive portfolio site. This site is fully responsive and the display will automatically adjust based on viewport width. The images are also responsive, so photos of different resolutions will download depending on viewport width. While the layout of my website is in HTML (no surprises there), I'm using jQuery for a couple of JavaScript features and Bootstrap for CSS.

Movie Trailer Site


This project uses a backend Python script to create a site with information about my favorite movies as well as the ability to watch their trailers from YouTube.

Portfolio Site


This project uses concepts in Udacity's Intro to HTML and CSS, Responsive Web Design Fundamentals, and Responsive Images courses to build a mock portfolio page. The page has both responsive design and images. The page dynamically resizes using Bootstrap and downloads the appropriate images based on viewport width.

Multi-User Blog


This project uses material from Udacity's Intro to Backend course to build a multi-user blog. Users can create accounts on the blog, post entries, and like and comment on other users' posts.

Tournament Database


This project uses concepts from Udacity's Intro to Relational Databases course to build a tournament database app. The database is for use in a Swiss-style tournament where competitors are matched based on similar win-loss records. The database schema uses PostgreSQL while the functions that interact with the schema are in a corresponding Python file.

Item Catalog


This project uses concepts from Udacity's Full Stack Foundations and Authentication and Authorization courses to build an item catalog web application. The app catalogs reading lists and books associated with each list. Users can log into the app using Google or Facebook account authorization, create reading lists and add books to the reading lists they create. In addition, the app also features JSON endpoints which allows users to access data in the catalog in a serialized format. The app uses the Flask development framework, SQLAlchemy, and Google and Facebook authorization APIs.

Neighborhood Map


This project uses concepts from Udacity's section on The Frontend: JavaScript & AJAX to build a mobile-friendly, single-page application featuring the Google Maps API. My map displays my favorite ice cream shops and tea cafes in the San Francisco Bay Area. Users can select a location from the map or from the displayed list, and users can also use the search box to filter the displayed list by location name, city, or neighborhood.

Server Configuration


The final project in the program uses concepts from the Deploying to Linux Servers section of Udacity's Full Stack Web Developer Nanodegree program to configure an Ubuntu server hosted on Amazon Lightsail. The server is secured from common attack vectors, contains a properly configured PostgreSQL database, and has my previously completed Item Catalog project running using Apache HTTP Server.
